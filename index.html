HTML

  • The head element contains information about the webpage, including the metadata, links, and title.
  • The body element represents the visible content shown to the user.
  • A header element within the body is labeled h1-6 as the title and subtitle.

CSS

  • There are three different ways to use CSS. Inline, Internal, and External.
  • The CSS link goes in the HTML head element.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Push your code to git often to save your work.
  • git status : checks what branch we are currently in.
  • git checkout -b : creates a new branch and switches to it.
  • Submiting a "pull request" is to send your code to be reviewed.
